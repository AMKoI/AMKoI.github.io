Диференціальні рівняння

Лінійні неоднорідні диференціальні рівняння другого порядку із постійними коефіцієнтами

Для вирішення неоднорідного рівняння достатньо знати загальне рішення однорідного рівняння (що просто зробити) та будь-яке окреме рішення неоднорідного. Розглянемо два особливі випадки правої частини неоднорідного рівняння та способи знаходження приватних рішень.

Перший випадок особливої правої частини – добуток багаточлена та показової функції:

Y + P Y + Q Y = P N ( x ) E (a x)

Де

S n ( x ) - багаточлен ступеня n з невизначеними коефіцієнтами;

E (a x) - та ж експонента, що й у правій частині рівняння;

X (r) - посилення.

Часткове рішення у цьому випадку шукатимемо у такому вигляді:

Y = S N ( x ) E (a x) X (r)

Розглянемо другий випадок особливої правої частини:

Y + P y + Q Y = E (a x) ( P N ( x ) cos b x + Q m ( x ) sin b x )

Часткове рішення у цьому випадку шукатимемо у такому вигляді:

Y = E (a x) ( S l ( x ) cos b x + T l ( x ) sin b x ) X (r)

Де

S l ( x ) , T l ( x ) - багаточлени ступеня ll з невизначеними коефіцієнтами, а l=max(n,m)l=max(n,m).
